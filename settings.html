Blue tabs

An efficiency-focused tab manager with integrated keyboard navigation provides a fast experience. All shortcuts are fully customizable, simply record your preferred keys below. Navigate with arrow keys or instantly target specific tabs using the assigned letter shortcuts (A-R) for quick switching and closing.

Key Names Keys Description
Extention shortcut Ctrl+Shift+X Open Blue tabs switcher
Switch + shortcut Space + A Switch to tab 1 (works for A-R keys)
Close + shortcut Shift + A Close tab 1 (works for A-R keys)
Close + Confirm Shift + Enter Close the currently highlighted tab
Confirm Enter Switch to selected tab or restore closed tab
Recently closed Tab Jump to recently closed tabs section
Settings Ctrl+Alt+S Open this settings page
